1. Introduction Of HTML

HTML stands for HyperText Markup Language.

It is the standard language used to create web pages.

HTML uses tags to structure content on a page.

2. Tag And Attributes

A tag is written inside angle brackets like <p>.

An attribute gives extra information to a tag.

Tag Attribute Example
<a> href, target Visit Google
<img> src, alt, width <img src="photo.jpg" alt="photo" width="100">
<p> id, class, title <p id="intro" class="text" title="tip">

4. Block vs. Inline Elements

Block elements take the full width and start on a new line:

This is a <div> — block element

This is a <p> — block element

Inline elements stay in the same line:

This is bold, this is italic, this is underlined, this is highlighted, this is a link — all inline, same line!

Block Elements Inline Elements
<div>, <p>, <h1>, <ul>, <table>, <section> <span>, <a>, <b>, <i>, <img>, <code>

5. Semantic Tags

Semantic tags describe the meaning of content.

Tag Purpose
<header>Top of the page — title and nav
<nav>Navigation links
<main>Main content area
<section>A group of related content
<article>Self-contained content like a blog post
<aside>Extra info, sidebar
<footer>Bottom of the page
<figure>Image or media with caption
<figcaption>Caption under a figure

Click to see: Why use semantic tags?

They help search engines, screen readers, and other developers understand your page better.
