1. Introduction of HTML

HTML stands for HyperText Markup Language.
It is the skeleton of every webpage — it tells the browser what content to show and how to structure it.

Basic HTML Document Structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Output:

Hello World!

2. Tags & Attributes

Tags are the building blocks of HTML.
Attributes add extra information inside the opening tag.

Syntax:

<tagname attribute="value"> Content </tagname>

Common Attributes Table:

Attribute Used On Purpose Example
id Any element Unique identifier <p id="intro">
class Any element Group elements <div class="box">
href <a> Link URL <a href="https://...">
src <img> Image file path <img src="photo.jpg">
alt <img> Fallback description <img alt="A cat">
width / height <img>, <table> Set size <img width="200">
target <a> Where to open link <a target="_blank">
title Any element Tooltip on hover <p title="tip">

Example — Anchor tag with multiple attributes:

<a href="https://example.com" target="_blank" title="Go to Example">Visit Example</a>

Output: Visit Example

4. Block vs. Inline Elements

Block elements take up the full width and always start on a new line.
Inline elements only take as much space as they need and stay in the flow of text.

Block Elements — each on its own row:

<div> — generic block container

<p> — paragraph block

<h4> — heading block

Common block tags:

TagDescription
<div>Generic block container
<p>Paragraph
<h1> – <h6>Headings
<ul> <ol>Lists
<table>Table
<section>Semantic section
<header>Page/section header
<footer>Page/section footer

Inline Elements — sit inside text:

This sentence has bold, italic, underline, highlight, code, a link, and a span — all inline, no line breaks.

Common inline tags:

TagDescription
<span>Generic inline container
<a>Hyperlink
<img>Image
<b> <strong>Bold
<i> <em>Italic
<code>Inline code
<br>Line break

5. Semantic Tags

Semantic tags tell the browser what the content means, not just how it looks.
They help with accessibility, SEO, and code readability.

Tag Purpose
<header>Top of page or section — logo, title, nav
<nav>Navigation links
<main>Main unique content of the page
<section>A thematic group of content
<article>Self-contained content (blog post, card)
<aside>Side content, tips, related info
<footer>Bottom of page — copyright, links
<figure>Image or diagram container
<figcaption>Caption for a figure
<time>Date or time value
<address>Contact information
<details>Expandable content block
<summary>Visible title for <details>
<mark>Highlighted / important text

Example — Semantic Page Structure:

<header>
  <h1>My Website</h1>
  <nav>
    <a href="#">Home</a> |
    <a href="#">About</a>
  </nav>
</header>

<main>
  <article>
    <h2>Post Title</h2>
    <p>Article content here...</p>
  </article>
  <aside>
    <p>Related links...</p>
  </aside>
</main>

<footer>
  <p>&copy; 2026 My Website</p>
</footer>

6. Lists

HTML has 3 types of lists:

A) Unordered List <ul> — bullet points

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Output:

  • HTML
  • CSS
  • JavaScript

B) Ordered List <ol> — numbered

<ol>
  <li>Learn HTML</li>
  <li>Learn CSS</li>
  <li>Build a project</li>
</ol>

Output:

  1. Learn HTML
  2. Learn CSS
  3. Build a project

C) Definition List <dl> — term + description

<dl>
  <dt>HTML</dt>
  <dd>Structure of the page</dd>
  <dt>CSS</dt>
  <dd>Style of the page</dd>
  <dt>JavaScript</dt>
  <dd>Behaviour of the page</dd>
</dl>

Output:

HTML
Structure of the page
CSS
Style of the page
JavaScript
Behaviour of the page

7. Images

The <img> tag is used to display images.
It is a self-closing (void) tag — it has no closing tag.
It is an inline element.

Important Attributes for <img>:

AttributeRequired?Purpose
srcYesPath or URL of the image
altYesText shown if image fails to load; used by screen readers
widthNoWidth in pixels
heightNoHeight in pixels
titleNoTooltip text on hover

Basic Image:

<img src="photo.jpg" alt="A beautiful mountain" width="300" />

Image from the Web:

<img src="https://via.placeholder.com/300x150" alt="Placeholder" width="300" />

Image with <figure> and <figcaption>:

<figure>
  <img src="sunset.jpg" alt="Golden sunset over hills" width="300" />
  <figcaption>A golden sunset over the hills</figcaption>
</figure>

Image as a Link:

<a href="https://example.com">
  <img src="logo.png" alt="Example logo" width="100" />
</a>

Topic Dictionary

Fun Facts

Did you know? — HTML was created in 1991

HTML was invented by Tim Berners-Lee in 1991. The first website ever made is still live at info.cern.ch!

Why is <alt> important?

The alt attribute helps visually impaired users understand images through screen readers. It also shows text when an image fails to load. Always write a meaningful alt!

Block vs Inline — quick trick to remember

Block = like a paragraph (takes the whole line).
Inline = like a word inside a sentence (shares the line).
